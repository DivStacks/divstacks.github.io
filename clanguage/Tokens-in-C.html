Important Questions.

Q: What is Token?

A: A token is an object that represents something else, such as another object (either physical or virtual), or an abstract concept as, for example, a gift is sometimes referred to as a token of the giver's esteem for the recipient. In computers, there are a number of types of tokens.

Q: What are C Tokens?

A: In C Language, the smallest individual units are called Tokens.

OR

It's a unit that makes sense to the compiler.

Q: What is Compiler?

A: A compiler is a computer program (or a set of programs) that transforms source code written in a programming language (the source language) into another computer language (the target language), with the latter often having a binary form known as object code.

Types Of Tokens.

There are various types of Tokens in C language, we are going to learn only three of them today.

1. Constants.

2. Variables.

3. Keywords.

1. Constants.

Q: What is Constant?

A: Something that does not or cannot change or vary is known as Constant.

OR

A quantity assumed to be unchanged throughout a given discussion is known as Constant.

Constants in C.

A constant refers to a value that does not change during the execution of a program. It is similar to mathematical calculations like "Pie" whose value is 22/7. Constants in C are divided in two main types:

1. Primary Constants.

2. Secondary Constants.

Today we will only talk about Primary Constants, As we have to deal with different things in our C programs like Numbers, Percentages or Alphabets. So on this behalf the Primary constants are divided in further types:

1. Integer Constants.

2. Real Constants.

3. Character Constants.

Let's take a look at them one by one.

Integer Constants.

C Integer Constant is a decimal (base 10), octal (base 8), or hexadecimal (base 16) number that represents an integer value, we use Integer Constants to represent integer values that cannot be changed.

OR

Integer constants are constants data elements that have no fractional parts or exponents.

Real Constants.

A real constant is combination of a whole numbers followed by a decimal point and the fractional part.

OR

Real constants are constants data elements that have some fractional parts or exponents.

Rules for constructing Real & Integer Constant.

1. It must have at least one digit.

2. Real constant must have a decimal point.

3. Integer constant must not have a decimal point.

4. It can be either negative or positive.

5. It is assumed to be positive if there is no sign specified.

6. Spaces and commas are not allowed within it.

Examples :

Some examples of Real constants are : 45.67, -6.55, 6663.9.

Some examples of Integer constants are : 426, +655, +31773, -7000.

Character Constants.

In addition to integers and real constants, the programs also need to deal with characters and strings. The character constants are used to express quantities like name, place or gender. They can be a single character or a group of characters.

Types of Character Constants:

1. Single Character Constants.

2. String Character Constants.

Single Character Constants :

It can be a single alphabet, a single digit or a single special symbol enclosed within single quotation marks. Both the inverted commas should point towards left. Some valid character constants are : 'a', 'b', '1', 'x'.

2. Variables:

Q: What is variable?

A: Something that does change or vary.

OR

A quantity assumed to be changed throughout a given discussion.

Variables in C:

A variable is a name, given to a temporary memory location that has been used to store any value. A variable is needed, when you want to store the input of user for calculation or you want to store a piece of output generated in the mid of calculation and to store the final output of the program.

Rules for Constructing Variable Names:

1. A variable is any combination of 1 to 31 alphabets, digits or underscore. This rule may vary with some compilers.

2. The first character in a variable name must be an alphabet or an underscore.

3. No commas or blanks are allowed within a variable name.

4. No special symbol other than an underscore ( _ ) is allowed.

5. A variable is declared by using a statement that contains the data type of the variable and the name of the variable. This statement can also contain initial value of a variable. For example, the statement "int age=18;" declares a variable age of integer type and initializes it to 18.

6. Since the maximum allowable length of variable name is 31 characters, an enormous number of variable names can be constructed using the above mentioned rules. It is a good practice to exploit this choice in naming variables by using meaningful variable names.

3. Keywords:

Keywords are the words whose meaning has already been explained to the compiler. The keywords cannot be used as variable names because if we do so, we are trying to assign a new meaning to the keyword, which is not allowed by the computer.
